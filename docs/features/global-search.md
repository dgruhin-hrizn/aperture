# Global Search

Search across your entire library to find movies, series, people, and more.

![Dashboard with Search](../images/features/dashboard.png)

## Accessing Search

- Click the **search icon** (üîç) in the top navigation bar
- On desktop: appears on the right side before your avatar
- On mobile: appears as an icon button

## Search Features

### Basic Search

Type any search term to find:

- **Movies** ‚Äî By title
- **TV Series** ‚Äî By title
- **People** ‚Äî Actors, directors, writers
- **Franchises** ‚Äî Movie collections and series

### Search Results

Results are grouped by type:

- **Movies** ‚Äî Shows poster, year, and rating
- **Series** ‚Äî Shows poster, year, network, and rating
- **People** ‚Äî Shows photo and known-for titles
- **Franchises** ‚Äî Shows collection name and item count

### Clicking Results

- **Movies/Series** ‚Äî Opens the detail page
- **People** ‚Äî Opens the person page with filmography
- **Franchises** ‚Äî Opens the franchise page

## Semantic Search

If your admin has enabled AI embeddings, search also uses **semantic matching**:

- Find content by description, not just title
- Search "mind-bending sci-fi" to find movies like Inception
- Search "heartwarming family films" for appropriate matches

## Search Tips

| Search For | Examples |
|------------|----------|
| Exact title | "The Dark Knight" |
| Partial title | "dark knight" |
| Actor name | "Tom Hanks" |
| Director | "Christopher Nolan" |
| Franchise | "Marvel" or "James Bond" |
| Description | "time travel romance" |

## What's NOT Searched

AI-generated libraries are **excluded from search** to keep results clean:

- AI Movie Recommendations library
- AI Series Recommendations library
- Top Picks libraries
- Shows You Watch libraries

You'll only see results from your main media libraries.
